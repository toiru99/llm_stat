# 벤치마크 (Benchmarks)

Artificial Analysis 리더보드에 쓰이는 벤치마크들의 OKF 개념 문서. 각 문서는 "무엇을 측정하나 / 핵심 사실 / 점수 해석 / 원본 논문"을 담는다. 원본 PDF는 [sources/](sources/).

# 종합 지수

* [Artificial Analysis Intelligence Index](aa-intelligence-index.md) - 9개 평가 가중 평균 종합 지능 점수 (v4.1)

# 지식·추론 (Scientific Reasoning & Knowledge)

* [Humanity's Last Exam (HLE)](humanitys-last-exam.md) - 수십 개 분야 전문가급 폐쇄형 지식·추론
* [GPQA Diamond](gpqa-diamond.md) - 박사급 과학 객관식, 구글 방어형
* [CritPt](critpt.md) - 현역 물리학자 출제, 연구 수준 물리 추론
* [AA-Omniscience](aa-omniscience.md) - 사실 회상 + 환각 회피 + 기권 (지식 신뢰성)

# 코딩 (Coding)

* [SciCode](scicode.md) - 실제 과학 연구 문제를 코드로 해결

# 에이전트 (Agentic)

* [GDPval](gdpval.md) - 실제 직종 전문가 실무 산출물 수행 (Real-World Work)
* [Terminal-Bench](terminal-bench.md) - 터미널/CLI에서 다단계 작업 자율 수행 (Hard·v2.1)
* [τ²-Bench (Telecom)](tau2-bench.md) - 이중 제어 환경 도구 사용·협업
* [τ³-Banking (τ-Knowledge)](tau3-banking.md) - 비정형 문서 검색 + 도구 기반 상태 변경
* [ITBench](itbench.md) - Kubernetes 인시던트 근본원인 분석 (SRE)
* [APEX-Agents](apex-agents.md) - 사무 도구 다루는 긴 호흡 교차 앱 업무

# 멀티모달 (Visual)

* [MMMU-Pro](mmmu-pro.md) - 텍스트만으론 못 푸는 멀티모달 다분야 이해·추론
