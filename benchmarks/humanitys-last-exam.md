# 무엇을 측정하나

수학·인문학·자연과학을 비롯한 **100여 개 분야의 전문가(대학원·박사급) 수준 폐쇄형 학술 지식과 깊은 추론**을 측정한다. 이름 그대로 "인류 최후의 시험(Humanity's Last Exam)" — 이런 종류의 폐쇄형 학술 벤치마크 중 **마지막**이 되는 것을 목표로 설계됐다.

배경은 명확하다. MMLU 같은 기존 인기 벤치마크에서 프런티어 LLM이 이미 90%+를 찍어 **포화**됐고, 그러면 최상위 모델 간 능력 차를 더는 가릴 수 없다. HLE는 그 천장을 다시 위로 올려, **현재 모델 능력과 인간 전문가 최전선 사이의 격차**를 정밀하게 재는 것이 목적이다.

# 과제 형식 / 예시

- **두 가지 문항 형식**:
  - **정확 일치형 단답(exact-match)**: 모델이 정확한 문자열을 출력 — 전체의 약 76%.
  - **객관식(MCQ)**: 5개 이상의 보기에서 하나 선택 — 약 24%.
- **멀티모달**: 약 **14%** 문항이 텍스트 + 이미지(도식·악보·비문·화학 구조 등)를 함께 이해해야 풀린다.
- 모든 문항은 **명확·검증 가능한 단일 정답**을 갖되, **검색으로는 답할 수 없게(non-searchable)** 설계 — 암기·단순 인출이 통하지 않는다.
- 분야 비중은 **수학 41%**로 가장 크고, 생물/의학 11%, CS/AI 10%, 물리 9%, 인문/사회 9%, 화학 7%, 공학 4%, 기타 9%. 특히 추론력을 광범위하게 시험하는 **세계 정상급 수학 문제**를 강조한다.
- 예시(논문 Figure 2): 팔미라 문자 비문 번역(고전학), 펑크터 간 자연 변환 개수 계산(수학), 그래프 마르코프 연쇄의 정상분포 수렴 조건(CS), 열적 페리고리 캐스케이드의 전자고리화 유형 판별(화학) 등 — 한 분야 박사라도 자기 세부 전공이 아니면 막히는 난이도.

# 채점 방식

- **정확도(accuracy)**가 1차 지표. 단답·객관식 모두 자동 채점에 적합하도록 답을 짧고 검증 가능하게 강제(소수↔분수 등 동등 표기 허용).
- 동시에 **RMS calibration error(보정 오차)**를 함께 보고한다. 모델에 답과 더불어 0~100% 확신도를 묻고, 확신도가 실제 정답률과 얼마나 어긋나는지를 측정 — **과신·환각의 척도**.
- 평가 시 표준 시스템 프롬프트로 "추론 후 최종답" 구조를 강제하고, 채점은 LLM 저지(o3-mini)로 동등 표기까지 맞춘다.

# 점수 해석 (높음 / 낮음)

- **높음** = 100여 개 분야를 가로지르는 전문가급 지식과 다단계 추론을 폭넓게 갖춘 **"올라운드 똑똑함"**. HLE는 워낙 어려워 점수 자체가 낮게 깔리므로, **몇 %p 차이도 큰 능력 차**를 의미한다.
- **낮음** = 폐쇄형 학술 최전선에서 무너짐. 다만 0%에 가까운 미세한 변동은 추론 노이즈일 수 있어 진척의 강한 신호로 보지 않는다(논문도 명시).
- **calibration error 높음** = 모르면서 자신 있게 틀린 답을 내놓음(환각 경향). 발표 당시 전 모델이 70%대 이상으로 매우 나빴다.

# 난이도 · 포화 · 현 SOTA

- 발표(2025) 시점 **사전공개 모델 정확도가 한 자릿수~10%대**에 머물렀다: GPT-4o **2.7%**, Grok 2 3.0%, Claude 3.5 Sonnet 4.1%, Gemini 1.5 Pro 4.6%, Gemini 2.0 Flash Thinking 6.6%, o1 **8.0%**, DeepSeek-R1 8.5%, o3-mini(high) **13.4%**.
- **calibration error는 전 모델 73~89%** — 능력 부족뿐 아니라 자기 한계 인식 실패(과신)까지 드러냈다.
- 즉 **미포화** 상태로 출발 → 상위 모델 간에도 변별력이 크다. (이후 추론 모델 세대가 올라오며 점수는 상승 추세지만, GPQA·MMLU처럼 천장에 붙진 않았다.)
- 출제 자체가 "프런티어 LLM을 틀리게 만든 문항만 통과"시키는 적대적 파이프라인이라, **설계상 어렵다**.

# 한계 · 주의

- **저점수 구간의 노이즈**: 점수가 낮을수록 추측·추론 변동의 영향이 커, 0% 근처 미세 변화는 진척으로 과대해석 금물(논문 명시).
- **폐쇄형(closed-ended) 한정**: 명확한 단일 정답이 있는 학술 문제만 — 개방형 추론·창작·장기 에이전트 능력은 측정 범위 밖. 출제 시 개방형·주관 해석·WMD 관련 문항은 배제.
- **분야 편중**: 수학 41%로 쏠려 있어 "전 분야 균등"으로 읽으면 안 됨. 추론(특히 수학적 추론) 비중이 크다.
- **객관식 추측 여지**: 24%는 5지+ 객관식이라 소폭의 추측 정답이 섞인다(논문도 비제로 정답률의 일부를 이 노이즈로 설명).
- **공개셋 게이밍·오염**: 2,500 공개셋은 학습 유입 가능성이 있어, **비공개 held-out 셋**으로 과적합·게이밍을 교차 검증한다.

# 데이터셋 구축 파이프라인

- **약 1,000명**의 분야 전문가(교수·연구자·박사학위자), **500개+ 기관 · 50개국**의 글로벌 협업.
- **70,000건 시도 → ~13,000건**(프런티어 LLM을 틀리게 만든 문항)만 전문가 리뷰로 진입 → **6,000 후보** → 조직위·전문가 승인 → **2,500 공개셋 + 비공개셋**.
- **2단계 전문가 리뷰**(반복 피드백 + 최종 승인), 표준 루브릭 적용. 양질 출제 유도를 위해 **$500,000 상금 풀**과 논문 공저 인센티브 운영.

# 다른 벤치마크와의 관계

- [[gpqa-diamond]] GPQA Diamond — 자연과학 3분야 박사급(좁고 깊음). HLE는 **분야가 훨씬 넓고 더 어렵다**(같은 추론 축).
- [[critpt]] CritPt — 물리 "연구 수준" 심층. HLE는 분야 폭으로 보완(같은 추론 축).
- [[mmmu-pro]] MMMU-Pro — 시각 다분야(멀티모달 축). HLE도 이미지 14%를 포함하나 주축은 추론·지식.
- MMLU — HLE가 명시적으로 "포화된 MMLU를 잇는 차세대 천장"으로 설계됨.

# 레이더 기여 근거

- **reasoning (1.0)**: 100여 분야를 가로지르는 다단계 추론이 본질이고, 수학(41%) 등 깊은 추론을 명시적으로 강조 → **추론 축 주 기여**.
- **knowledge (0.3)**: 폭넓은 전문 지식도 요구하지만, 설계 핵심이 "검색 불가 + 깊은 추론"이라 순수 사실 지식 폭은 부차적 → **전문 지식 축에 낮은 가중치로 부 기여**. (GPQA의 knowledge 0.4보다 낮게 둔 것은, HLE가 지식 회상보다 추론 쪽으로 더 치우쳐 설계됐기 때문.)

# 한국 · 앨런 관점

가장 어려운 종합 지식·추론 시험이라, 모델이 **전문가급 질의에서 어디까지 믿을 만한지의 상한선**을 가늠하는 축이다. 특히 **calibration error**는 "모델이 모르는 걸 모른다고 하는가"를 직접 보여줘, 환각에 민감한 한국어 전문 상담·연구 보조 용도에서 신뢰성 판단의 보조 지표가 된다. 점수가 낮게 깔려 모델 간 변별력이 커서, 최상위 모델을 고를 때 유용하다.
