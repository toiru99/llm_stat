# 능력 영역 (Capability Domains)

"이 LLM은 어떤 영역을 잘하는가"의 핵심 매핑. AA 리더보드가 각 벤치마크에 붙인 능력 영역을 한국어 카테고리로 묶었다. 모델별 영역 강약을 볼 때는 해당 영역의 벤치마크 점수들을 함께 본다.

# 지식·추론 (Knowledge & Reasoning)

깊은 지식과 논리적 추론. 사실성·신뢰성 포함.

* [HLE](/benchmarks/humanitys-last-exam.md) - 종합 전문가급 지식·추론
* [GPQA Diamond](/benchmarks/gpqa-diamond.md) - 과학(생물·물리·화학) 추론
* [CritPt](/benchmarks/critpt.md) - 물리 연구급 추론
* [AA-Omniscience](/benchmarks/aa-omniscience.md) - 지식 정확도 + 환각 회피(신뢰성)

# 코딩 (Coding)

* [GDPval](/benchmarks/gdpval.md) - 실제 직무 산출물 (Real-World Work)
* [Terminal-Bench](/benchmarks/terminal-bench.md) - 터미널/CLI 코딩 (Agentic Coding & Terminal Use)

* [ITBench](/benchmarks/itbench.md) - K8s 장애 근본원인 분석 (SRE/DevOps)
